# Skyline ACE: Visual Overdrive 기능 구현 현황

## ✅ 구현 완료 기능 (Implemented)

### 1. 시각적 특수 효과 (Visual Effects)
- [x] **여객기 비행운 (Contrail):** 고고도 여객기 뒤에 생성되는 실시간 비행운 파티클 효과
- [x] **강화된 폭발 효과:** 적기 격추 시 섬광 파티클과 확산되는 회색 연기(Smoke) 효과 적용
- [x] **야간 시인성 개선:** 밤 시간대에도 전장과 유닛을 밝게 유지하는 조명 및 발광(Emissive) 설정

### 2. 전장 환경 및 시뮬레이션
- [x] **이륙 시퀀스 (Takeoff):** 활주로 가속 및 자동 기수 상승(Lift-off) 연출
- [x] **이동하는 군함 & 여객기:** 바다와 하늘의 오브젝트들이 실제 속도로 이동하며 맵 순환
- [x] **무한 루프 맵:** 맵 끝 도달 시 반대편으로 연결되는 루프형 비행 구조

### 3. 레이더 및 HUD
- [x] **정밀 원형 레이더:** 플레이어 중심 5km 범위 내 적군 유닛 실시간 탐지 및 표시
- [x] **통합 HUD:** 고도, 속도, 처치 수, 제한 시간 정보 표시
- [x] **가감속 레버:** 스로틀 출력 제어 및 엔진 사운드 동적 피치 연동

### 4. 사운드 시스템 (Sound)
- [x] **동적 엔진음:** 스로틀 출력에 연동된 엔진 사운드
- [x] **전투 효과음:** 레이저 사격 및 적기 격추 시 폭발음 적용
- [x] **배경음:** 비행 몰입을 위한 배경 앰비언트 사운드
