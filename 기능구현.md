# Skyline ACE: Global Takeoff 기능 구현 현황

## ✅ 구현 완료 기능 (Implemented)

### 1. 이륙 및 비행 시스템
- [x] **이륙 시퀀스 (Takeoff):** 활주로 대기 -> 가속 -> 자동 기수 상승(Lift-off) 연출
- [x] **무한 루프 맵:** 맵 경계 도달 시 반대편으로 연결되는 순환형 비행 구조
- [x] **자동 비행 안정화:** 조종 입력 없을 시 기체가 자동으로 수평을 유지
- [x] **가감속 레버:** 스로틀 레버를 통한 정밀 출력 제어 및 사운드 피치 연동

### 2. 사운드 시스템 (Sound)
- [x] **동적 엔진음:** 스로틀 출력에 따라 변하는 엔진 사운드
- [x] **전투 효과음:** 레이저 발사음 및 적기 격격추 시 폭발 효과음
- [x] **배경음악:** 게임 몰입을 위한 배경 앰비언트 사운드 적용

### 3. 전투 및 AI
- [x] **난이도별 미션:** 5단계 난이도 및 목표 처치 수(10*난이도) 설정
- [x] **적/아군 AI:** 플레이어를 추적하는 적기 및 아군 유닛 자동 출격
- [x] **자동 조준:** 난이도에 따른 확률적 명중 시스템

### 4. 환경 및 그래픽
- [x] **활주로 구현:** 항공모함 데크 및 활주로 모델링
- [x] **시간대 설정:** 실제 시간에 따른 낮/밤 자동 설정 (밤에도 밝은 시인성)
- [x] **지형:** 바다 위 빌딩 숲과 함선들이 배치된 복합 지형
