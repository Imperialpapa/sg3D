# Skyline ACE: Global Landmarks 기능 구현 현황

## ✅ 구현 완료 기능 (Implemented)

### 1. 콘텐츠 및 월드 시뮬레이션
- [x] **세계 랜드마크:** 에펠탑, 피라미드, 자유의 여신상 등 주요 건축물 3D 모델링 및 배치
- [x] **아군 편대 비행:** 플레이어 좌우에서 대형을 유지하며 비행하는 아군 AI 시스템
- [x] **이동하는 군함 & 여객기:** 바다와 하늘의 오브젝트들이 실제 속도로 이동하며 맵 순환
- [x] **비주얼 강화:** 여객기 비행운(Contrail) 및 강화된 파티클 폭발 효과

### 2. 레이더 및 HUD 확장
- [x] **다중 탐지 레이더:** 적군(빨강), 아군(초록), 랜드마크(흰색)를 구분하여 실시간 표시
- [x] **통합 HUD:** 고도, 속도, 처치 수, 제한 시간 정보 표시
- [x] **가감속 레버:** 스로틀 출력 제어 및 엔진 사운드 동적 피치 연동

### 3. 이륙 및 비행 시스템
- [x] **이륙 시퀀스 (Takeoff):** 활주로 가속 및 자동 기수 상승(Lift-off) 연출
- [x] **무한 루프 맵:** 맵 끝 도달 시 반대편으로 연결되는 루프형 비행 구조
- [x] **자동 비행 안정화:** 조종 입력 없을 시 기체가 자동으로 수평 유지

### 4. 사운드 시스템 (Sound)
- [x] **동적 엔진음:** 스로틀 출력에 연동된 엔진 사운드
- [x] **전투 효과음:** 레이저 사격 및 적기 격추 시 폭발음 적용
