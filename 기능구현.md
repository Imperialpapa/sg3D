# Skyline ACE: Flight Calibration 기능 구현 현황

## ✅ 구현 완료 기능 (Implemented)

### 1. 비행 물리 및 조작 고도화
- [x] **지수적 감도 조이스틱:** 입력값에 곡선을 적용하여 미세 조작과 급기동을 동시에 지원
- [x] **자동 수평 유지 (Auto-Leveling):** 조작 중단 시 쿼터니언 기반의 안정적인 수평 복귀 로직
- [x] **선회(Yaw) 물리 통합:** 롤(Roll) 조작 시 자연스러운 방향 전환이 이루어지는 비행 물리 시스템
- [x] **이륙 시퀀스:** 활주로 가속 및 자동 이륙 연출

### 2. 콘텐츠 및 월드 시뮬레이션
- [x] **세계 랜드마크:** 에펠탑, 피라미드, 자유의 여신상 배치
- [x] **아군 편대 비행:** 플레이어를 호위하며 대형을 유지하는 아군 AI 시스템
- [x] **이동하는 군함 & 여객기:** 바다와 하늘의 동적인 오브젝트 이동 및 순환

### 3. 레이더 및 HUD
- [x] **다중 탐지 레이더:** 적군(빨강), 아군(초록), 랜드마크(흰색) 실시간 탐지
- [x] **가감속 레버:** 정밀한 스로틀 제어 및 엔진 사운드 피치 연동

### 4. 사운드 및 이펙트
- [x] **동적 사운드:** 엔진, 레이저, 폭발 효과음 적용
- [x] **시각 효과:** 여객기 비행운 및 강화된 파티클 폭발 이펙트
