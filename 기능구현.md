# Skyline ACE: Global Takeoff 기능 구현 현황

## ✅ 구현 완료 기능 (Implemented)

### 1. 전장 환경 및 시뮬레이션
- [x] **이륙 시퀀스 (Takeoff):** 활주로 가속 및 자동 이륙 연출
- [x] **이동하는 군함:** 바다 위 함선들의 항진 및 순환 시스템
- [x] **여객기 비행:** 고고도 여객기 출현 및 직선 비행 효과
- [x] **무한 루프 맵:** 맵 경계 도달 시 반대편으로 자연스럽게 연결되는 비행 구조

### 2. 레이더 및 HUD
- [x] **정밀 원형 레이더:** 플레이어 중심 5km 범위 내 적군 유닛 실시간 탐지 및 표시
- [x] **통합 HUD:** 고도, 속도, 처치 수, 제한 시간 정보를 직관적으로 표시
- [x] **가감속 레버:** 스로틀 출력에 따른 정밀 속도 제어 및 엔진 사운드 피치 연동

### 3. 사운드 시스템 (Sound)
- [x] **엔진 사운드:** 출력(Thrust)에 따라 실시간으로 변하는 엔진음
- [x] **효과음:** 레이저 사격음 및 격추 시 폭발음 적용
- [x] **배경음:** 비행 몰입을 위한 배경 앰비언트 사운드

### 4. 전투 및 AI
- [x] **난이도 시스템:** 5단계 스테이지 및 목표 처치 수 설정
- [x] **적군 추격 AI:** 플레이어를 감지하고 추격하는 적 기체 시스템
- [x] **자동 조준:** 난이도 기반 확률적 명중 보정
